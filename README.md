## 🚀 快速开始

### 在线阅读

👉 **[https://openclaw-guide-zh.pages.dev](https://openclaw-guide-zh.pages.dev)**

### 本地阅读

```bash
git clone https://github.com/aaronyang-ai/openclaw-guide-zh.git
cd openclaw-guide-zh
npm install
npm run dev
```

浏览器打开 `http://localhost:4321` 即可阅读。
